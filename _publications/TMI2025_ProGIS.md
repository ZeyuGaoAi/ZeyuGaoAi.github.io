excerpt: 'Interactive segmentation is attractive for computational pathology because it can reduce annotation cost while still accommodating pathologist input. ProGIS introduces a prototype-guided interactive segmentation framework that segments pathological structures accurately with minimal interaction and can recover multiple same-type connected components from one prompt. This makes slide-level interactive pathology segmentation more efficient and practical than fully automatic or heavily interactive alternatives.'

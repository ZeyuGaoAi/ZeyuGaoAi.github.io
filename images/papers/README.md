# Paper Figure Drop Folder

Put publication main figures in this folder.

Recommended naming rule:

- `CVPR2026_CARE.png`
- `MedIA2026_PH2ST.png`
- `MedIA_Megaseg.png`
- `MedIA_StaDis.png`
- `TMI2022.png`
- `TMI2023.png`
- `TMI2023_MGTrans.png`
- `TMI2023_SGMF.png`
- `TMI2025.png`
- `MICCAI2020.png`
- `MICCAI2021_0.png`
- `MICCAI2021_1.png`
- `MICCAI2024.png`

Preferred format:

- `png`
- horizontal figure if possible
- main model / overview figure works best

Once figures are added here, they can be linked in `_publications/*.md` using `header.teaser`.
